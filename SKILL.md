# 股票综合分析框架 V1.0

> 融合两套系统精华：
> - Hermes量价分析：深度K线量价关系解读、5源新闻情绪、主力行为逐K线追踪
> - daily_stock_analysis：可插拔策略框架、7条核心理念、评分体系、大盘复盘、风控机制

---

## 一、7条核心交易理念（不可违反）

1. **严进策略** — 不追高，乖离率(MA5偏离)<5%才考虑入场
2. **趋势交易** — MA5>MA10>MA20多头排列，顺势而为
3. **效率优先** — 关注筹码结构好、资金配合的票
4. **买点偏好** — 优先在MA5/MA10附近回踩买入，不追突破
5. **风险排查** — 高风险利空一票否决，宁错过不做错
6. **量价配合** — 成交量必须验证价格运动，无量上涨=虚假
7. **龙头放宽** — 真正的板块龙头可适当放宽标准，但必须有辨识度

## 二、分析流程（5步法）

### 步骤1: 量价关系分析（核心，第一优先）

**目标**: 通过逐日K线+成交量判断主力处于什么阶段

**主力行为4阶段识别框架**:

| 阶段 | 量价特征 | 典型K线 |
|------|---------|--------|
| 建仓期 | 缩量阴跌后放量阳线；底部温和放量价不跌 | 底部长下影阳线 |
| 洗盘期 | 缩量回调（量缩价不跌）；地量出现=洗盘尾声 | 缩量十字星 |
| 拉升期 | 温和放量上涨（量价齐升）；放量突破平台 | 放量中阳线 |
| 出货期 | 天量天价；放量下跌；冲高回落上影线长 | 高位大阴线 |

**⚠️ 每个判断必须附带具体K线实例**:
- 日期 + 开/收/高/低
- 成交量 + 与前日对比(放大/缩小百分比)
- 为什么这根K线说明主力在做X

**量价背离（趋势拐点预警）**:
- 顶背离：价格创新高但成交量未创新高 → 见顶信号
- 底背离：价格创新低但成交量未创新低 → 见底信号

**量价评分**: 1-5分
- 5分: 底部放量建仓+量价同步上涨
- 4分: 缩量企稳+温和放量反弹
- 3分: 量价中性，无明确信号
- 2分: 有出货迹象但未完全确认
- 1分: 天量天价/放量暴跌/量价全面恶化

### 步骤2: 资金流向验证

**目标**: 用资金流数字验证量价判断是否准确

- 主力资金连续净流入 → 印证建仓/拉升判断
- 主力资金连续净流出 → 印证出货判断
- 量价说建仓但资金在流出 → 判断可能有误，需重新审视

**关键规则**: 资金流是辅助验证，量价才是主力真实意图

### 步骤3: 技术面+均线系统

**均线多空排列判断**:
- 多头排列: MA5>MA10>MA20>MA30 → 趋势向上
- 空头排列: MA5<MA10<MA20<MA30 → 趋势向下
- 纠缠态: 均线交叉缠绕 → 方向不明

**关键技术指标**:
- MACD: 金叉/死叉、零轴上下
- RSI: 超买(>70)/超卖(<30)
- 量比: >2放量 / <0.5缩量
- 换手率: >10%异常活跃 / <1%冷门

### 步骤4: 估值+基本面

**估值检查**:
- PE(TTM) vs 行业均值
- PB vs 每股净资产
- PEG = PE / 净利润增速(%) → <1便宜 / 1-1.5合理 / >1.5贵
- 前向PE = 当前价 / 机构一致预期EPS

**基本面关注**:
- 净利润增速（同比）
- ROE水平
- 机构覆盖度（有无近期研报）
- 机构一致预期EPS

### 步骤5: 消息面+风险筛查

**消息面情绪打分**:
- 利好/利空条数统计
- 关键新闻重要性评估
- 消息面总情绪: 偏多/中性/偏空 + 得分

**风险筛查（RiskAgent模式）**:
必须检查的风险项:
- [ ] 大股东减持/质押
- [ ] 业绩预亏/变脸
- [ ] 监管处罚/立案调查
- [ ] 行业政策利空
- [ ] 限售解禁（30天内）
- [ ] 估值异常（PE>100或为负、PB>10）
- [ ] 技术破位（死叉、跌破关键支撑）

**风险分级**:
- high: 重大风险（造假/诉讼/大规模减持）→ 一票否决
- medium: 显著风险（业绩不及预期/解禁/板块利空）→ 降级信号
- low: 轻微风险（分析师下调/小额减持）→ 记录但不阻断

## 三、决策评分体系

### 综合评分权重
| 维度 | 权重 | 说明 |
|------|------|------|
| 量价关系 | 30% | 主力行为阶段+量价评分 |
| 技术面 | 25% | 均线排列+趋势强度+关键位 |
| 资金面 | 20% | 主力资金流向验证 |
| 消息面 | 15% | 情绪+催化+风险 |
| 估值面 | 10% | PE/PEG/基本面安全边际 |

### 评分→操作映射
- 80-100分: 🟢 买入/加仓（条件全满足，高确信）
- 60-79分:  🟢 买入/加仓（多数积极，小瑕疵）
- 40-59分:  🟡 持有/观望（信号混合或有风险）
- 20-39分:  🔴 卖出/减仓（趋势偏空+风险）
- 0-19分:   🔴 卖出/止损（高风险+看空）

### 操作性护栏（Actionability Guardrails）
- **不因单日涨跌在买/卖之间反复跳** — 至少观察2-3日确认趋势
- **买入需支撑位确认** — 或有效突破阻力位+量能/资金流验证
- **卖出需破支撑** — 或主力持续出逃、风险明显升高
- **价格在支撑与阻力之间且资金流不明** → 偏好中性操作(hold/watch)
- **高风险一票否决** — 任何high级风险直接降为卖出

## 十、Pitfalls

### 绝不编造数据点
支撑位/压力位必须来自实际K线数据(mootdx bars)。禁止推测。
用户会追问"这个数据哪里来的"——答不上来就是假数据。

### 量价分析必须有K线实例
每个主力行为判断必须附带具体哪天的K线+量价数据。
不能只说"主力在出货"而不说"5/27放量长上影阴线，成交量放大14%..."

### 资金流数字只是验证
量价关系是判断主力行为的第一依据，资金流数字是辅助验证。
两者矛盾时优先信量价。

### 加仓决策不能只看估值
PE便宜不等于可以加仓。必须同时满足量价+技术+资金+估值。
"估值合理但主力在出货"= 不能加仓。

### 消息面利好发布时机
注意利好新闻是否配合股价高位发布——可能是出货掩护。
5/27深科技"利好"发布恰在最高点47.03，就是典型案例。
