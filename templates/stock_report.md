== 量价关系分析(核心) ==

  主力行为阶段: [建仓/洗盘/拉升/出货/做T/阴跌]
  量价评分: X/5

  阶段划分(近30日):
    阶段1 (日期范围): [描述]
    阶段2 (日期范围): [描述]

  关键K线(判断依据):
    ★ K线1: 日期
      开/收/高/低 + 成交量(vs前日%)
      → [这根K线说明主力在做什么，为什么]

    ★ K线2: ...
    ★ K线3: ...

== 核心数据 ==
  当前价/涨跌/PE/PB/市值/换手率/量比

== 消息面 ==
  情绪: 偏多/中性/偏空(得分: X.XX)
  利好X条/利空X条
  关键新闻: [1-3条]

== 资金面(验证量价) ==
  20日主力净流入: X.XX亿
  近5日逐日流向摘要
  验证结论: [印证/矛盾 + 说明]

== 技术面 ==
  MA5/10/20/30 + 偏离%
  均线排列: 多头/空头/纠缠
  MACD/RSI状态
  关键支撑/阻力位

== 风险筛查 ==
  [逐项列出检查结果]

== 综合评分 ==
  量价X分×30% + 技术X分×25% + 资金X分×20% + 消息X分×15% + 估值X分×10%
  = XX分

== 操作建议 ==
  已持有 → [建议]
  未持有 → [建议]
  仓位: X成
  止损位: XX.XX

== 加仓信号(如适用) ==
  ✅/❌ × 5项 → X/5
  结论: 🟢/🟡/🔴
