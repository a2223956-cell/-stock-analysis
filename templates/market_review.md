# 大盘复盘模板

```
== 大盘复盘: {日期} ==

  【趋势结构】
    上证/深证/创业板是否同向: ✅/❌
    量能: 放量/缩量/持平
    关键支撑/阻力是否被突破

  【资金情绪】
    涨跌家数比: X:X
    涨停X家/跌停X家
    成交额: X亿(vs前日%)

  【主线板块】
    领涨: [板块名] — 催化因素
    领跌: [板块名] — 原因

  【市场定性】
    进攻(共振上行) / 均衡(分化震荡) / 防守(转弱减仓)

  【明日策略】
    仓位建议: X成
    关注方向: [板块/题材]
    风控要点: [注意事项]
```
