# 腾讯行情API字段解析参考

## API地址
```
https://qt.gtimg.cn/q=sz001203,sh600552,...
```
- 深市前缀: `sz` (000xxx, 002xxx, 300xxx)
- 沪市前缀: `sh` (600xxx, 601xxx, 688xxx)
- 批量查询用逗号分隔，比单只查询更稳定

## 字段结构(以~分隔)

响应格式: `v_sh600552="1~凯盛科技~600552~23.37~24.13~...";`

### 核心字段位置(固定)

### ⚠️ 关键陷阱

1. **[9]-[28]是买卖五档**，共20个字段，会干扰手动数字段
2. **[31]-[34]是涨跌额/幅/高/低**，这是最可靠的解析锚点
3. **成交额在[37]，单位是万元**，除以10000=亿元
4. **PE在[39]**，但注意有些票该字段可能为空

### 批量查询推荐解析方式

```python
import urllib.request

codes = 'sz002979,sh600552,sz000021'
url = f'https://qt.gtimg.cn/q={codes}'
resp = urllib.request.urlopen(url)
raw = resp.read().decode('gbk')

segments = raw.split('v_')
for seg in segments:
    if '~' not in seg:
        continue
    f = seg.split('~')
    if len(f) < 45:
        continue
    name = f[1]
    code = f[2]
    price = f[3]
    prev = f[4]
    open_p = f[5]
    change_pct = f[32]    # 涨跌幅%
    high = f[33]           # 最高
    low = f[34]            # 最低
    amount_wan = f[37]     # 成交额(万元)
    turnover = f[38]       # 换手率%
    amplitude = f[43]      # 振幅%
    pe = f[39] if f[39] else 'N/A'
    pb = f[46] if len(f) > 46 and f[46] else 'N/A'
    
    amount_yi = round(float(amount_wan) / 10000, 2)
    print(f'{name}({code}) 收:{price} 涨跌:{change_pct}% 成交:{amount_yi}亿 换手:{turnover}% PE:{pe}')
```
