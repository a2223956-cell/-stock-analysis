# 利好兑现出货案例分析 (2026-07-15~7/20)

## 定义
"利好兑现出货"是指：重大利好公告发布当天=股价全天最高价，随后主力借利好人气大规模出货，股价连续暴跌。

与"涨停出货"的区别：
- 涨停出货: 涨停板次日高开低走（1天延迟）
- 利好兑现出货: 利好发布当天即见顶（0延迟），后续多日连续暴跌

## 案例: 通富微电(002156) H1业绩预增288-337%

### 时间线
```
7/14下午: 发布H1业绩预增288-337%公告
7/14收盘: 77.87(全天最高,看涨锤子线)
7/15开盘: 84.10(跳空高开!) → 最高84.70 → 收盘78.71
7/16: 75.95(-3.5%)
7/17: 68.36(-10.0%,收盘=最低)
7/20: 63.00(-7.84%,天量134亿)
→ 从84.70→63.00, 3天暴跌-25.6%!
```

### V2报告失误分析(80.5分→-25.6%)
V2报告(7/15盘中14:06)评分80.5分看多，核心错误：
1. 将"业绩预增288%"视为持续利好 → 实际利好已price in
2. 天量293万手+价格84.70被忽视 → 经典天量天价见顶
3. "回踩79.38加仓" → 加仓=被套(现在63元)
4. 止损77.87太紧 → 7/17已击穿

### 识别信号
1. 利好公告发布当天的K线特征：
   - 高开(利用FOMO情绪)
   - 冲高回落(上影线长)
   - 天量成交(主力趁机出货)
2. 次日验证：
   - 如果次日收阴+放量 = 利好兑现出货确认
   - 如果次日继续涨 = 利好尚未充分定价(罕见)

### 量化判断
- 利好发布日成交额 vs 前5日均值
  - 如果发布日成交额 > 前5日均值×2 = 高概率出货
  - 通富微电: 7/14成交178亿 vs 前5日均值145亿 = 1.23倍(偏高)
- 利好发布后3日跌幅
  - 如果>10% = 利好兑现出货确认

### 教训
1. 利好发布当天=全天最高价 是最常见模式
2. 天量天价是"利好兑现出货"的最佳确认信号
3. 评分系统应将"利好刚发布+天量"标记为高风险
4. 盘中"锤子线"必须收盘确认,不能在盘中判定

## 与通富微电79.39案例的关系
两个案例都涉及通富微电,但问题不同：
- 79.39案例: K线数据编造(LLM幻觉)
- 利好兑现出货: 判断逻辑错误(确认偏误+锚定效应)
