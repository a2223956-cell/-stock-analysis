# 通富微电V2报告失败案例研究 (2026-07-15→7/20)

## 事故概述

7/15V2报告评分80.5/100(偏多/买入),建议"回踩79.38加仓,止损77.87"。
此后3个交易日股价暴跌-19.2%(84.70→68.36),止损位被大幅击穿。

## V2报告5项具体错误

| # | V2判断 | 实际结果 | 错误类型 |
|---|--------|---------|---------|
| 1 | "突破前高79.38,空间打开" | 84.70后连跌3天至68.36 | 假突破误判 |
| 2 | "业绩预增288-337%持续催化" | 利好兑现后主力出货 | 利好兑现出货 |
| 3 | "回踩79.38加仓" | 79.38已被跌穿至68.36 | 加仓=被套 |
| 4 | "止损77.87" | 7/17收盘68.36已破止损 | 止损位被大幅击穿 |
| 5 | 7/15天量293万手="突破确认" | 天量天价=经典见顶信号 | 天量天价遗漏 |

## 根因分析

### 1. 确认偏误 (Confirmation Bias)
看到H1业绩预增+突破前高 → 偏向看多 → 忽视出货信号

### 2. 锚定效应 (Anchoring)
被"业绩预增288%"锚定 → 未考虑利好已price in
(7/9-7/14已从65涨至78,+20%,利好已被市场消化)

### 3. 天量天价遗漏
最重要的见顶信号(天量+天价)被忽视:
- 7/15成交量293万手 = 30日最大
- 7/15价格84.70 = 30日最高
- 经典"天量天价"=主力在最高点大量出货

### 4. 止损位设置不当
77.87(7/14收盘)在高波动股上太紧
应参考MA10或更宽的止损区间

## "利好兑现出货"识别框架

### 典型模式
```
利好发布 → 股价当天暴涨 → 次日天量天价 → 连续下跌
```

### 识别信号(缺一不可)
A. 利好发布后1-2天出现30日天量+30日天价
B. 利好发布后资金并未持续流入
C. 股价已从底部上涨>30%
D. 同板块其他标的未同步上涨

### 通富微电时间线
```
7/9:  涨停72.17(+10%) → 拉升启动
7/14: V形反转77.87 → H1业绩预增288-337%发布
7/15: 天量293万+84.70(30日最高) = 见顶!
7/16: 75.95(回落)
7/17: 68.36(暴跌-10%) → 资金-15.40亿
7/20: 68.60 → 从高点跌-19.2%
```

## 评分趋势教训

通富微电4次评分: 70→38.5→80.5→33.5
- 震荡模式 = 分析框架对该股不稳定
- 操作难度大,应降低仓位或回避

## 改进措施

1. 利好发布后等待1-2天,观察是否出现天量天价
2. 天量天价是最高优先级的见顶信号,不能被利好叙事覆盖
3. 评分剧烈震荡的股票应降低仓位
4. 高波动股止损位应设在MA10附近而非收盘价
5. "利好兑现"≠"持续催化",需区分时机
