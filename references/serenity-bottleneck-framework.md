# Serenity瓶颈投资法 — A股应用框架

> 来源: 美股传奇散户Serenity的"紫苏叶理论"
> 适用: 产业链上游瓶颈标的筛选(物理AI/机器人/半导体等)

## 核心思想

"走窄门" — 在AI等大趋势中，不买最显眼的热门股，而是沿着产业链
向下钻，寻找未来架构迁移中最难替代的瓶颈点，并在旧估值/旧偏见
仍压制价格时提前下注。

## 五因子选股模型

```
确定需求 × 受限供给 × 低关注度 × 价值捕获 × 催化剂
```

| 因子 | 含义 | A股筛选标准 |
|------|------|------------|
| 确定需求 | 趋势是否不可逆 | 行业政策支持+全球龙头布局 |
| 受限供给 | 是否难以替代 | 国内少数厂商/技术壁垒高/认证周期长 |
| 低关注度 | 市场是否充分定价 | 市值<100亿/无机构深度覆盖/近期无大涨 |
| 价值捕获 | 公司能否吃到肉 | 有真实订单/送样/量产/收入占比提升 |
| 催化剂 | 什么事件能触发重估 | 新客户/量产突破/政策落地/龙头采购 |

## 六步研究流程

1. **找大趋势** — 确定不可逆的产业方向(如物理AI/具身智能)
2. **画产业链地图** — 拆解感知层→决策层→执行层→载体层
3. **找瓶颈** — 哪个环节供给最受限、最难替代
4. **找证据** — 客户网站/合作公告/财报措辞/送样记录
5. **做风控** — 如果客户不放量/竞争替代/估值透支怎么办
6. **匹配仓位** — 只看别人总结的→小仓位; 自己研究的→可大仓位

## 已知局限

1. **推断过拟合** — 你觉得是瓶颈,市场可能不买账
2. **早期估值无锚** — 小公司PE参考意义有限
3. **流动性风险** — 小盘股进出冲击大
4. **产业验证周期长** — 送样→量产→放量可能1-2年
5. **幸存者偏差** — Serenity的成功案例是幸存者

## A股特殊风险

- 概念炒作后迅速退潮(参考万向钱潮-21%)
- 小盘股波动极大(参考蠡湖股份-18%)
- 同板块内分化严重(有催化的涨,没催化的跌)

## 实战案例: 光刻胶产业链选股(2026-06-30)

### 产业链地图

```
上游原料                中游制造                配套材料              设备
├─ 彤程新材(树脂龙头)   ├─ 晶瑞电材(g/i线)     ├─ 安集科技(CMP液)    └─ 奥普光电(光学)
├─ 圣泉集团(酚醛树脂)   ├─ 南大光电(ArF唯一!)   ├─ 鼎龙股份(CMP垫)
└─ 东材科技(聚酰亚胺)   ├─ 上海新阳(ArF研发中)  ├─ 飞凯材料(湿化学品)
                        └─ 彤程新材(KrF/ArF扩产)
```

★ 瓶颈判定: ArF光刻胶(先进制程必需,国内仅1家量产) + CMP抛光液(国内1家垄断)

### 五因子筛选结果(12→4只)

| 排名 | 名称 | 市值 | PE | 五因子 | 入选理由 |
|------|------|------|-----|--------|---------|
| 1 | 安集科技 | 737亿 | 88.5 | 22/25 | CMP垄断+已量产+估值相对低 |
| 2 | 南大光电 | 592亿 | 179 | 21/25 | ArF唯一量产,但估值透支 |
| 3 | 奥普光电 | 134亿 | 137 | 21/25 | 最小市值+光刻机概念,但价值捕获弱 |
| 4 | 彤程新材 | 563亿 | 96.5 | 20/25 | 树脂龙头+ArF扩产,PEG 2.14最合理 |

### 深度分析结论

- **安集科技(首选)**: Serenity 23/25, CMP瓶颈完全成立, PE 88 < 同行鼎龙96, 主升浪初期+超大单流入
- **奥普光电(不建议)**: 价值捕获仅2/5, 光刻机供应未确认+股东减持窗口, R:R 1:1.15
- **南大光电(观察)**: 瓶颈最强但PE 179无法介入, R:R 1:0.05几乎无上行空间
- **彤程新材(次选)**: PEG 2.14最合理, 但连续涨停后RSI 83超买, 等回调

### 关键教训

1. **瓶颈≠可以买**: 南大光电瓶颈最强(ArF唯一量产),但PE 179倍→R:R极差,不能买
2. **价值捕获是核心因子**: 奥普光电概念好但"最后一公里"未打通(供应关系未确认)→不值得下注
3. **估值要用同行对比**: 安集PE 88看似高,但鼎龙PE 96→龙头反而更便宜,这是相对价值
4. **板块分析必须区分个股vs板块**: 光刻胶板块今日整体涨,但个股强弱差异大(安集+6.3% vs 飞凯-3.6%)

## 板块选股标准流程(Serenity六步法应用)

当用户要求"用Serenity选XX板块的股票"时:

1. **画产业链地图**: 拆解上游→中游→下游→配套,标注每个环节的竞争格局
2. **找瓶颈**: 哪个环节供给最受限、技术壁垒最高、国产化率最低
3. **获取板块成分股**: 东财概念板块API / web_search
4. **五因子初筛**: 需求×供给×低关注×价值捕获×催化 → 打分排序,取前3-4名
5. **深度分析**: 对入选票做完整量价+资金+估值+消息面分析
6. **横向对比+排序**: 最终推荐排序+操作建议+止损位

⚠️ 注意: 五因子初筛只是缩小范围,最终决策必须看量价关系和R:R比
   高分但R:R差的票(如南大光电)不能买
